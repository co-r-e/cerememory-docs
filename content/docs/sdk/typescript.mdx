## Installation

```bash
npm install @cerememory/sdk
```

Or with yarn:

```bash
yarn add @cerememory/sdk
```

## Client Setup

```typescript
import { CerememoryClient } from "@cerememory/sdk";

const client = new CerememoryClient({
  baseUrl: "http://localhost:8420",
  apiKey: "your-api-key",  // optional
  timeout: 30_000,         // ms
});
```

## Methods

### store

Store a new memory record.

```typescript
const recordId = await client.store({
  text: "The user prefers TypeScript over JavaScript for new projects.",
  store: "semantic",
  emotion: "trust",
  metadata: { source: "conversation", topic: "preferences" },
});

console.log(recordId);  // UUID string (e.g., "01916e3a-5678-7000-8000-000000000001")
```

The high-level `store()` method returns the UUID string directly. For the full response including `store` and `initial_fidelity`, use the protocol-level `encodeStore()` method.

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `text` | string | Yes | -- | Text content |
| `store` | StoreType | No | auto | Target store |
| `emotion` | string or EmotionVector | No | undefined | Emotion label or vector |
| `embedding` | number[] | No | undefined | Pre-computed embedding |
| `metadata` | Record | No | undefined | Arbitrary metadata |
| `sessionId` | string | No | undefined | Session context |

### storeBatch

Store multiple records in a single request.

```typescript
const result = await client.storeBatch({
  records: [
    { text: "First memory", store: "episodic" },
    { text: "Second memory", store: "episodic" },
  ],
  inferAssociations: true,
});

console.log(result.results.length);        // 2
console.log(result.associationsInferred);  // number of inferred links
```

### recall

Query memories with a text cue.

```typescript
const memories = await client.recall({
  text: "What does the user prefer for new projects?",
  limit: 10,
  mode: "human",
  stores: ["semantic", "episodic"],
  minFidelity: 0.1,
  reconsolidate: true,
});

for (const memory of memories) {
  console.log(`[${memory.record.store}] score=${memory.relevance_score.toFixed(4)}`);
  console.log(`  fidelity=${memory.record.fidelity.score.toFixed(4)}`);
}
```

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `text` | string | Yes | -- | Text query cue |
| `limit` | number | No | 10 | Maximum results |
| `mode` | "human" or "perfect" | No | "human" | Recall mode |
| `stores` | StoreType[] | No | all | Filter by store |
| `minFidelity` | number | No | undefined | Minimum fidelity |
| `reconsolidate` | boolean | No | true | Strengthen recalled memories |
| `activationDepth` | number | No | 2 | Association traversal depth |

### recallAssociate

Traverse the association graph from a specific record.

```typescript
const associations = await client.recallAssociate(
  "01916e3a-5678-7000-8000-000000000001",
  {
    types: ["semantic", "temporal"],
    depth: 3,
    min_weight: 0.2,
    limit: 20,
  }
);

for (const memory of associations.memories) {
  console.log(`${memory.record.id} via ${memory.activation_path.join(" -> ")}`);
}
```

### forget

Permanently delete records.

```typescript
await client.forget({
  recordIds: ["01916e3a-5678-7000-8000-000000000001"],
  cascade: false,
  confirm: true,  // required
});
```

### stats

Get system statistics.

```typescript
const stats = await client.stats();
console.log(`Total records: ${stats.total_records}`);
console.log(`Avg fidelity: ${stats.avg_fidelity.toFixed(3)}`);
console.log(`By store:`, stats.records_by_store);
```

### introspectRecord

Get full details for a specific record.

```typescript
const record = await client.introspectRecord("01916e3a-5678-7000-8000-000000000001");
console.log(`Fidelity: ${record.fidelity.score}`);
console.log(`Noise: ${record.fidelity.noise_level}`);
console.log(`Access count: ${record.access_count}`);
```

### lifecycleDecayTick

Trigger a manual decay cycle.

```typescript
const result = await client.lifecycleDecayTick({ tick_duration_seconds: 3600 });
console.log(`Updated: ${result.records_updated}, Pruned: ${result.records_pruned}`);
```

### lifecycleConsolidate

Trigger memory consolidation.

```typescript
const result = await client.lifecycleConsolidate({
  strategy: "incremental",
  min_age_hours: 24,
  min_access_count: 3,
  dry_run: true,
});
```

## Error Handling

```typescript
import {
  CerememoryError,
  RecordNotFoundError,
  RateLimitedError,
} from "@cerememory/sdk";

try {
  await client.introspectRecord("nonexistent-id");
} catch (error) {
  if (error instanceof RecordNotFoundError) {
    console.error(`Not found: ${error.recordId}`);
  } else if (error instanceof RateLimitedError) {
    console.error(`Rate limited, retry after ${error.retryAfter}s`);
  } else if (error instanceof CerememoryError) {
    console.error(`CMP error [${error.code}]: ${error.message}`);
  }
}
```
