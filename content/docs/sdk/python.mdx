## Installation

```bash
pip install cerememory
```

For the native in-process binding, see [Native Bindings](/docs/sdk/native-bindings).

## Sync Client

```python
from cerememory import Client

client = Client(
    "http://localhost:8420",
    api_key="sk-cerememory-prod-abc123",
    timeout=30.0,
    max_retries=0,
    retry_mutating_requests=False,
)
```

<Callout type="info">
Retries are conservative by default in `0.2.2`: `max_retries` defaults to `0`, and mutating requests are retried only when `retry_mutating_requests=True`.
</Callout>

## High-Level Methods

<Callout type="info">
The high-level helpers are text-first conveniences. For multimodal blocks, manual associations, full encode context, or custom `MemoryContent`, use the protocol-level methods such as `encode_store()`.
</Callout>

### `store(text, ...)`

Store a text memory and receive its UUID.

```python
record_id = client.store(
    "The user's preferred editor is Neovim.",
    store="semantic",
    metadata={"source": "onboarding"},
)
print(record_id)
```

Supported keyword arguments:

| Argument | Type | Default | Description |
|----------|------|---------|-------------|
| `store` | `str | StoreType` | `StoreType.EPISODIC` | Target store |
| `emotion` | `EmotionVector | None` | None | Optional emotion vector |
| `metadata` | `dict | None` | None | Stored with the record |

### `recall(query, ...)`

Recall memories matching a text cue.

```python
memories = client.recall(
    "What editor does the user prefer?",
    limit=5,
    stores=["semantic", "episodic"],
    mode="human",
    reconsolidate=True,
    activation_depth=2,
)

for memory in memories:
    print(memory.record.store, memory.relevance_score)
```

| Argument | Type | Default | Description |
|----------|------|---------|-------------|
| `limit` | `int` | `10` | Maximum number of results |
| `stores` | `list[str | StoreType] | None` | None | Restrict the search |
| `mode` | `str | RecallMode` | `RecallMode.HUMAN` | `human` or `perfect` |
| `min_fidelity` | `float | None` | None | Minimum fidelity threshold |
| `reconsolidate` | `bool` | `True` | Strengthen recalled memories |
| `activation_depth` | `int` | `2` | Spreading activation depth |

### `forget(*record_ids, ...)`

```python
deleted = client.forget(record_id, confirm=True, cascade=False)
print(deleted)
```

### `store_raw(text, ...)`

Store a verbatim raw journal entry.

```python
raw_id = client.store_raw(
    "The user said they prefer Neovim",
    session_id="sess_001",
    topic_id="editor-preferences",
    source="conversation",
    speaker="user",
)
```

| Argument | Type | Default | Description |
|----------|------|---------|-------------|
| `session_id` | `str` | -- | Required. Session identifier |
| `topic_id` | `str | None` | `None` | Topic identifier for dream grouping |
| `source` | `str | RawSource` | `"conversation"` | `conversation`, `tool_io`, `scratchpad`, `summary`, or `imported` |
| `speaker` | `str | RawSpeaker` | `"user"` | `user`, `assistant`, `system`, or `tool` |
| `visibility` | `str | RawVisibility` | `"normal"` | `normal`, `private_scratch`, or `sealed` |
| `secrecy_level` | `str | SecrecyLevel` | `"public"` | `public`, `sensitive`, or `secret` |

### `recall_raw(...)`

Query the raw journal for verbatim entries.

```python
entries = client.recall_raw(
    query="editor preference",
    session_id="sess_001",
    limit=10,
)

for entry in entries:
    print(entry.session_id, entry.speaker)
```

| Argument | Type | Default | Description |
|----------|------|---------|-------------|
| `query` | `str | None` | `None` | Full-text search query |
| `session_id` | `str | None` | `None` | Filter by session |
| `limit` | `int` | `10` | Maximum results |
| `include_private_scratch` | `bool` | `False` | Include `private_scratch` entries |
| `include_sealed` | `bool` | `False` | Include `sealed` entries |
| `secrecy_levels` | `list[str | SecrecyLevel] | None` | `None` | Restrict to specific secrecy levels |

### `dream_tick(...)`

Summarize raw journal entries into curated episodic and semantic memories.

```python
result = client.dream_tick(session_id="sess_001")
print(result.episodic_summaries_created, result.semantic_nodes_created)
```

| Argument | Type | Default | Description |
|----------|------|---------|-------------|
| `session_id` | `str | None` | `None` | Process only entries from this session |
| `dry_run` | `bool` | `False` | Preview without creating records |
| `max_groups` | `int` | `10` | Maximum topic groups to process |
| `include_private_scratch` | `bool` | `False` | Include `private_scratch` entries |
| `include_sealed` | `bool` | `False` | Include `sealed` entries |
| `promote_semantic` | `bool` | `True` | Promote factual content to the semantic store |
| `secrecy_levels` | `list[str | SecrecyLevel] | None` | `None` | Restrict to specific secrecy levels |

### `stats()`, `get_record()`, `health()`, `readiness()`

```python
stats = client.stats()
record = client.get_record(record_id)
health = client.health()
readiness = client.readiness()
```

`stats()` returns `background_decay_enabled`, `background_dream_enabled`, raw journal counts, dream processing status, and optional `evolution_metrics` in addition to the usual record counters.

Example:

```python
metrics = client.introspect_evolution()
print(metrics.parameter_adjustments)
```

## Async Client

The async client mirrors the sync API:

```python
from cerememory import AsyncClient

async with AsyncClient("http://localhost:8420") as client:
    record_id = await client.store("Async memory", store="episodic")
    memories = await client.recall("async memory")
    await client.forget(record_id, confirm=True)
```

Use `AsyncClient` for asyncio-based services, agents, or web backends.

## Error Handling

```python
from cerememory import (
    CerememoryError,
    RecordNotFoundError,
    RateLimitedError,
    UnauthorizedError,
)

try:
    client.get_record("00000000-0000-0000-0000-000000000000")
except RecordNotFoundError:
    print("Record not found")
except RateLimitedError as exc:
    print(exc.retry_after)
except UnauthorizedError:
    print("Missing or invalid API key")
except CerememoryError as exc:
    print(exc.code, exc.message)
```
